## Docker Image with node & awscliv2

The image was created to be used in GitLab CI/CD. It uses the following versions:

- Docker 24.0.2-cli
- Node.js 14.21.3
- AWS CLI v2 (latest)

Published to Docker Hub with the tag `neobility/docker:24.0.2-cli-node-awscliv2`.
